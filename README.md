# QR Code Generator

A simple QR Code Generator web app that allows users to:

- Generate QR codes instantly
- Add a custom logo/image inside the QR
- Download QR as PNG image
- Create QR for links, text, names, messages, etc.
- Use on both desktop and mobile devices
